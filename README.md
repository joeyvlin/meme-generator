# Meme Generator

A React-based meme generator that allows you to create memes by adding text overlays to images.

## Features

- Upload your own images or select from predefined templates
- Add unlimited text overlays
- Drag text to reposition on the image
- Resize text using slider or input field
- White text with black border (classic meme style)
- Download your meme as PNG

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Usage

1. Select an image by uploading your own or choosing from templates
2. Click "Add Text" to add a text overlay
3. Click and drag text to reposition it
4. Use the slider or input to adjust font size
5. Edit text content in the text input field
6. Click "Download Meme" to save your creation
